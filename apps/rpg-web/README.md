# @lunchtable-rpg/web

RPG creator + library + session client for LunchTable RPG.

## Runtime

- Default API base: `http://localhost:3210`
- Override with env: `VITE_RPG_API_BASE`

## Start

```bash
bun run dev:rpg
```

## Feature Surface

- Library: list/search/featured worlds
- Creator: create world draft, bootstrap flagship worlds, generate campaign
- Session: create/join/fetch/action live sessions, dice rolling (remote-first, local fallback)
