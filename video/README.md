# Promo Video (Skill-based)

This folder follows the `create-promo-video` skill pattern:
- Remotion composition in `src/`
- Portrait format for short-form platforms
- Zod schema + editable text props in `src/Root.tsx`

## Render

```bash
cd video
bun install
bun run render
```

Output:
- `out/promo-portrait.mp4`

## Edit copy/colors interactively

```bash
cd video
bun run dev
```
